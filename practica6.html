Figura compleja de manera dinamica usando el elemento Canvas

¿Que es canvas?

El elemento HTML canvas se utiliza para dibujar gráficos, sobre la marcha, a través de JavaScript. El elemento canvas es solo un contenedor de gráficos. Debe usar JavaScript para dibujar los gráficos. Canvas tiene varios métodos para dibujar rutas, cuadros, círculos, texto y agregar imágenes.

Un canvas es un área rectangular en una página HTML. De forma predeterminada, un canvas no tiene borde ni contenido.
Se denotan asi: < canvas id="myCanvas" width="200" height="100"> < /canvas>

Siempre hay queespecificar un atributo "id" (al que se hará referencia en un script) y un atributo "width y height" para definir el tamaño del canvas.
